# Amazon Financial Statement and Valuation Analysis

## Overview
This project provides a comprehensive financial analysis of Amazon Inc. using financial analytics, ratio analysis, discounted cash flow (DCF) valuation, competitor benchmarking, and industry analysis techniques.

## Objective
The goal of this project is to evaluate Amazon's:
- Financial performance
- Liquidity position
- Profitability trends
- Debt management
- Competitive positioning
- Intrinsic valuation

## Tools & Techniques Used
- Microsoft Excel
- Financial Statement Analysis
- Ratio Analysis
- Discounted Cash Flow (DCF) Valuation
- Relative Valuation
- SWOT Analysis
- Industry Benchmarking
- Financial Analytics

## Areas Analyzed

### Financial Statements
- Balance Sheet Analysis
- Income Statement Analysis
- Revenue Growth Analysis
- Profitability Analysis

### Ratio Analysis
- EBITDA Margin
- Quick Ratio
- Current Ratio
- Interest Coverage Ratio
- PP&E Turnover Ratio
- Book Value Per Share

### Valuation Techniques
- Discounted Cash Flow (DCF)
- Relative Valuation
- Price-to-Earnings (P/E)
- EBITDA Multiple
- Sales Multiple

### Competitor Analysis
Competitors analyzed:
- eBay
- Wayfair
- ODP Corporation
- Stitch Fix

## Key Findings
- Amazon demonstrated strong long-term revenue growth and market dominance.
- AWS and diversification strategies significantly strengthened profitability.
- DCF valuation indicated strong long-term growth potential.
- Relative valuation suggested potential market undervaluation.
- Rising WACC and macroeconomic risks created short-term margin pressure.

## Business Impact
This project demonstrates practical applications of:
- Corporate finance analysis
- Equity valuation
- Financial forecasting
- Competitive benchmarking
- Investment decision-making
